Spring Boot Employee Management System

A complete REST API built using Spring Boot for managing employee data.
This project follows a clean layered architecture (Controller → Service → Repository) and includes DTO mapping, validation, exception handling, and MySQL integration.

 Features
 
Create new employee

Update employee

Fetch employee by ID

Fetch all employees

Delete employee

DTO for request/response

Global exception handling

Service-layer business logic

MySQL database connection

Technologies Used

Java 21

Spring Boot

Spring Web

Spring Data JPA

MySQL

Spring devTool

Lombok

Maven

REST best practices

Auto-validation using @Valid
